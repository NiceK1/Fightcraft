## File Requirements:
- **Format**: PNG with transparency support
- **Size**: 64x64 pixels (recommended)
- **Naming**: Use lowercase with underscores (as shown above)
- **Quality**: Keep file sizes reasonable for game performance

## Usage:
The game will automatically load these PNGs instead of generating colored rectangles for materials. If a PNG is missing, it will fall back to the colored rectangle.
